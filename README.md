<h1>Requirements:</h1>
- Android 7 and higher
<br>
- Termux
<br>
- Termux X11
<h1> 1. Update your packages:</h1>
pkg update -y && pkg upgrade -y
<h1> 2. Clone repo:</h1>
git clone https://github.com/mmmagneummm/Termux-Xfce.git
<h1> 3. Enter in repo directory:</h1>
cd Termux-Xfce
<h1> 4. Run install script:</h1>
sh install.sh
